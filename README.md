# Smart Contract Auction

A decentralized auction system built using Solidity.
This project was developed as part of the Coursera Smart Contracts course.

## Features
- Bidder self-registration
- Token based bidding system
- Multiple items auction
- Random winner selection
- Owner-only winner reveal (modifier used)

## Tech Stack
- Solidity
- Remix IDE
- Ethereum JavaScript VM

## How to Run
1. Deploy the contract in Remix IDE
2. Register bidders using multiple accounts
3. Place bids on different items
4. Reveal winners (only owner can call)
5. Check winners using the winners() function
